System Implementation

MVC and .NET Web Solution for Adelante Hispanic Achievers
CIS 420

After gathering system requirements in CIS 320, in CIS 420 our teams are tasked with implementing a solution. The team I am a part of decided to use MVC and .NET, which is not the usual route groups in this class take. Normally, groups decide to use WordPress, which is also a suitable solution. However, my group wanted a challenge and to push ourselves to expand our .NET skills.

Our technology choices have several benefits for Adelante. Using MVC .NET as the development backbone reduces the amount of code that is directly tied to the user interface, which enhances the testability of the application. This means that in the future, the system solution can be easily customized and tested for functionality. Bootstrap, CSS, HTML, and JavaScript will allow our developers to provide Adelante with a customized and responsive user interface. Coding the responsiveness and functionality on our own prevents the use of plugins from unknown, external developers, which improves the security of the system. Depending on external developers’ code or relying on their security patches is a reactive strategy that fails to acknowledge today’s information security challenges. This system will improve functionality and will enhance Adelante’s ability to communicate with and superintend its users, manage grants, generate reports and securely store data.

Our website was not selected as the winner of the project, but we were still very proud of our accomplishment of building this site.
